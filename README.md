# SKF Assignment

## Description

Home assigment - tree menu fetch from Rest API

## Installation

To install json server

`npm install -g json-server`

To install

`npm install`

To run

`npm start`

`npm run server`

## To build

`npm run build`

Then use the dist folder you get.
